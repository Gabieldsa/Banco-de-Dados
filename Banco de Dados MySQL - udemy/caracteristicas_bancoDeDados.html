Características de um banco de dados

Bancos de dados devem executar procedimentos chamados de transações. A integridade de uma transação deve ser regida por quatro propriedades:

  • Atomicidade

    • Todas as ações devem ser concluídas com sucesso, ou o processo falha como um todo e toda a ação é desfeita (rollback). Se há sucesso em todas as ações a informação é mantida no banco (commit);


  • Consistência

    • Deve-se obedecer regras e restrições definidas em um banco, como por exemplo uso de chaves estrangeiras ou uso de campos únicos;


  • Isolamento

    • Cada transação deve ser independente de outras transações;


  • Durabilidade

    • Os resultados de uma transação devem ser permanentes, exceto se outra transação a desfizer.

Descrição

MySQL é um dos sistemas de gerenciamento de bancos de dados mais utilizados em todo mundo. Este curso irá lhe ensinar a criar bancos de dados, atualizar tabelas, fazer buscas em bases de dados, e muito mais. Aprenda a utilizar a ferramenta MySQL Workbench e a controlar bancos de dados em seu browser usando o PHPMyAdmin. Matricule-se já!

O que você aprenderá

  • Aulas práticas de bancos de dados
  • Teoria da linguagem SQL
  • Sistema de Gerenciamento Banco de Dados MySQL
  • Projetar bases de dados e construir tabelas
  • Inserir, atualizar e remover dados
  • Criar usuários e gerenciar permissões em bases de dados
  • Gerenciar bancos de dados usando MySQL Workbench
  • Gerenciar bancos de dados usando PHPMyAdmin
  • Instalar e configurar MySQL em um servidor Linux
  • Manipular bases de dados via terminal
  • Realizar backup de bases de dados MySQL
  • MySQL Intermediário

Há algum requisito ou pré-requisito para o curso?

  • Conhecimentos básicos em computação (instalar programas, criar pastas e acessar sites)

Para quem é este curso:

  • Pessoas interessadas em aprender sobre bancos de dados
  • Estudantes de computação
  • Desenvolvedores web (PHP e JavaScript)
  • Desenvolvedores Python, Perl, C, C++ e Java
  • Pessoas com sede de conhecimento ;)
